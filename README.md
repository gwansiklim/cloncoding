## 프로젝트 내용
___

이번 프로젝트는 클론코딩을 한다. 우리조는 페이스북 클론코딩을 하기로 정하였다. 리액트 2명 노드 2명 프론트 2명 백 2명이
페이스북의 기능을 최대한 따라 해보려고 한다.

## 기능 구현
___
* 로그인 / 회원가입 - 본인
* 게시글 CRUD - 백앤드 팀장

## 로그인 / 회원가입시 사용한 패키지
___
* express
* mysql, mysql2
* sequelize
* sequelize-cli
* joi 
* jsonwebtoken
